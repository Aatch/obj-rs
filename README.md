obj-rs [![cargo-i][]][cargo-a] [![travis-i][]][travis-a]
========

[Wavefront obj][obj] parser for **[Rust][]**. It handles both `.obj` and `.mtl` formats.

```toml
[dependencies]
obj-rs = "*"
```
